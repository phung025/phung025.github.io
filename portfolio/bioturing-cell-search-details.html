BioTuring Cell Search

Project information

  • Category: Algorithm & backend service
  • Location: BioTuring | Vietnam
  • URL: https://bioturing.com/
  • 03/2020 - 08/2020

A key task in single-cell analysis is cell type annotation, which often requires researchers to manually compare the gene expression of selected cell populations against established marker genes for each cell type. To streamline this process, BioTuring Cell Search algorithm leverages the curated and well-annotated cell populations in the BioTuring database. By identifying populations within this database that have gene expression profiles similar to the queried population, the algorithm can accurately identify the cell type of an unknown population.
